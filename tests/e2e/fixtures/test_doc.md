# Aurora Lens Calibrator

The Aurora Lens Calibrator is a fictional bench instrument used for testing the
RAG pipeline. It is not based on a real person, real facility, real device, or
private operational procedure.

## Safety Envelope

The calibrator is operated only in a synthetic lab scenario. The bench must be
clear of reflective samples, the mock aperture door must remain closed, and the
training operator records only fictional run identifiers. If the status strip
shows amber, the scripted procedure stops and the operator resets the simulated
interlock.

## Assembly Sequence

The reference rail attaches to the left bracket before the lens cradle is
seated. The fictional torque dial should be set to mark three, and the alignment
flag should point toward the printed grid. The fixture card is then placed under
the cradle so the mock sensor can read its orientation.

## Calibration Marker

The acceptance check depends on a single color fact: the calibration marker is
cobalt blue. The backup marker is intentionally ignored during the acceptance
check, and the training note says not to substitute green, amber, or violet for
the cobalt blue marker.

## Verification Table

After setup, the operator reads three fictional fields. The rail field should
show `aligned`, the cradle field should show `seated`, and the marker field
should show `cobalt blue`. A run is accepted only when all three fields match
the synthetic manual.

## Troubleshooting

If the fictional sensor reports a marker mismatch, the operator repeats the
assembly sequence and inspects the fixture card. If the rail field remains
unaligned, the mock bracket is reseated. If the marker field remains wrong, the
procedure records a synthetic failure code and does not continue.
