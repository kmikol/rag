# Orbital Seed Catalog

The Orbital Seed Catalog is a fictional reference sheet used only for the
end-to-end RAG test. It does not describe a real facility, real crop, real
mission, or private operating procedure.

## Catalog Purpose

The catalog records synthetic seed packets for a mock orbital greenhouse. Each
packet has an invented tray position, a fictional humidity band, and a training
beacon that helps the test ask a distinct question from the lens calibrator
fixture.

## Recovery Beacon

The recovery beacon has one required phrase: the seed catalog recovery tone is
silver chime. The training note says not to substitute bronze hum, violet pulse,
or amber bell for the silver chime recovery tone.

## Packet Ledger

The mock packet named Mariner Basil sits in tray seven. The mock packet named
Polar Mint sits in tray two. The mock packet named Harbor Dill sits in tray
five. The recovery operator records `silver chime` beside every accepted packet.

## Verification Notes

This document intentionally repeats the synthetic recovery tone in multiple
sections so retrieval has a clear, citable source after ingestion. The terms are
invented for testing and should not be interpreted as real agricultural or
mission guidance.
