# Running

Runtime commands will be added when the first service skeleton is implemented.

The initial service set is expected to be:

- `api-service`
- `ingestion-worker`
- `embedding-service`
- `postgres`
- `qdrant`
- `ollama`

The backend should be configurable through environment variables documented in [Useful Commands](useful-commands.md) and [ADR 008](../adr/008-job-coordination-and-service-contracts.md).
